Reports a non-optional substitution ${path} whose root key exists in the config but whose full path does not — a likely typo, such as ${app.hsot} where app exists but app.hsot does not.

Substitutions rooted at a key that is not defined anywhere in the config (for example ${ENV}) are deliberately left alone, because they usually resolve from system properties or environment variables at runtime. The quick fix converts the substitution to an optional ${?path}, which silently yields no value when unresolved.
